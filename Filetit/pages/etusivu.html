Opiskelijaravintolat

Ruokalistat, jotka löytyvät yhdestä paikasta

Opis Ravintolat tarjoaa selkeän pohjan päivittäisille ja viikoittaisille ruokalistoille. Kun API liitetään myöhemmin, sama näkymä voi näyttää oikeat ravintolat ilman rakenteen vaihtamista.

Selaa ravintoloita Katso kartta
Päivän menu Nopea näkymä tämän päivän lounaaseen.
Viikon menu Sama rakenne toimii koko viikon ruokalistalle.
Karttanäkymä Lähin ravintola korostuu ja paikat löytyvät nopeasti.

Miksi tämä toimii

Selkeä rakenne myöhempää APIa varten

Tässä vaiheessa sivusto on jo käyttökelpoinen esittelyyn ja jatkokehitykseen.

Yksi data-kerros

Ravintolat ja menut vaihdetaan yhdestä paikasta

JS ei ole sidottu yhteen sivuun, vaan sama data ruokkii listat, kartan ja nostot.

Helppo laajentaa

Suodattimet, valinta ja aikajakso on jo valmiina

Kun oikea REST API lisätään, käyttäjälogiikka voi pysyä samana.

Siisti julkaisu

Responsiivinen ja luettava myös puhelimella

Ulkoasu käyttää selkeitä kortteja, kontrastia ja joustavaa gridiä.

Esimerkki

Nostoja lähellä olevista ravintoloista

Valmiina API:ta varten

Rakenne on tehty jatkoa varten, ei vain demoa varten

Ruokalista-, kartta- ja yhteystietonäkymät on pilkottu niin, että myöhempi REST API voi täyttää sisällöt ilman HTML-remonttia.
